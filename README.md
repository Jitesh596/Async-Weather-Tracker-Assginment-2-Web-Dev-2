🌤 #Async Weather Tracker


🚀 ##Features

🔍 Search weather by city name

🌡️ Display temperature, weather condition, humidity, and wind speed

⚡ Uses async/await with Fetch API

🛑 Handles invalid city errors

💾 Stores search history using Local Storage

🔁 Click previous searches to fetch weather again

🧠 Displays Event Loop console logs (sync, microtask, macrotask)

🛠 ##Tech Stack

HTML5

CSS3

JavaScript

Fetch API

Local Storage
